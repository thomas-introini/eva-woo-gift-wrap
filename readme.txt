Adds a "Confezione regalo" (gift wrap) option to the WooCommerce Checkout Block with a fixed €1.50 fee.

== Description ==

EVA Gift Wrap is a lightweight plugin that adds a gift wrapping option to the WooCommerce Checkout Block.

When customers enable the option, a fixed €1.50 fee is added to their order.

**Features:**

* Simple checkbox in the checkout totals area
* Fixed €1.50 non-taxable fee
* Works exclusively with the WooCommerce Checkout Block
* Clean, minimal codebase
* Italian labels (Confezione regalo)

**Requirements:**

* WordPress 6.0 or higher
* WooCommerce 8.0 or higher
* WooCommerce Blocks (included in WooCommerce)
* PHP 8.0 or higher

== Installation ==

1. Upload the `eva-gift-wrap` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The gift wrap option will automatically appear on checkout pages using the Checkout Block.

== Frequently Asked Questions ==

= Does this work with the classic checkout shortcode? =

No, this plugin only works with the WooCommerce Checkout Block.

= Can I change the fee amount? =

Currently the fee is fixed at €1.50. Future versions may include a settings page.

= Is the fee taxable? =

No, the gift wrap fee is not taxable by default.
